1. (0,50) Explique que vantagens e desvantagens existem no fato do JavaScript ser uma linguagem interpretada pelo navegador.
Resposta:
Vantagens:
Execução do Lado do Cliente: Alivia a carga no servidor e proporciona uma experiência ágil para o usuário.
Interatividade: Permite interfaces de usuário dinâmicas e responsivas.
Desvantagens:
Desempenho Limitado: Pode ser inferior em aplicações intensivas em processamento, apesar das melhorias contínuas.
Visibilidade do Código: Exposto aos usuários, o que pode representar riscos de segurança.


2 (1.00)Associe os eventos com suas respectivas funções:

Resposta: onclick: Ocorre quando o elemento HTML é clicado.
onkeypress: Ocorre quando uma tecla é pressionada enquanto o elemento HTML tem o foco.
onblur: Ocorre quando o elemento HTML perde o foco.
onfocus: Ocorre quando o elemento HTML recebe o foco.
onsubmit: Ocorre quando um formulário é submetido.
onload: Ocorre quando a página HTML é completamente carregada.
onchange: Ocorre quando o valor de um elemento HTML é alterado (pode ser usado em campos de formulário, por exemplo).
onmousemove: Ocorre quando o mouse é movimentado sobre o elemento HTML.
